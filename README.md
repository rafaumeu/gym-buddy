## 🚀 Quick Start Guide

### Prerequisites

- 🔧 Node.js 16.0.0+
- 🐳 Docker
- 🐘 Docker Compose

### Installation Magic

```bash
# Clone the repository
git clone https://github.com/rafaumeu/gym-buddy
cd gym-buddy

# Install dependencies
npm install

# Start PostgreSQL
docker-compose up -d

# Run database migrations
npx prisma migrate dev

# Launch in development mode
npm run start:dev
```
